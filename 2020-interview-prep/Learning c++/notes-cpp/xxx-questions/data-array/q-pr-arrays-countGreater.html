C++: Arrays: Programming Problem - Count Greater

Name __________________________________

(10 points) Write a function, countGreater, to count the number of values in an array that are larger than a specified floating=point value. The prototype is 

int countGreater(float val, float b[], int n);
  • val is the value to compare against each of the array values in b.
  • b is an array of n floating-point numbers b[0]..b[n-1].
  • n is the number of elements to examine in the b array.

This is an example main program that uses the countGreater function. 

int main() {
   float nums[] = {3.5, 1.0, 1.5};
   float wips[] = {2000., 300., -555, 0.0, 1.9999};
   int count;
   
   count = countGreater(1.0, nums, 3);
   cout << count << endl;  // prints 2
   
   count = countGreater(-33.3, wips, 5);
   cout << count << endl;  // prints 4

   return 0;
}

You should write only the function. The function, as usual, should do no I/O.

Answer: 
int countGreater(float val, float b[], int n) {
    int count = 0;
    for (int i=0; i<n; i++) { // go thru each element
        if (b[i] > val){      // compare it to val
            count++;          // count if greater
        }
    }
    return count;
}
