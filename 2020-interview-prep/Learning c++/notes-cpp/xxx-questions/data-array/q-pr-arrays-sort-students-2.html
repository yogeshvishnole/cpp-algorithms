This is an extension of the Sort Students Problem and a sample Sort Students Solution, make these modifications.

  • Data. Read the data from a text file. Notes were handed out on how to do this. The data is in the file test-data.txt, which you can see by clicking on that link and then copying and pasting into a file, or by right-clicking the link and saving it as a file.
  • Sort the students by last name, and if the last names are equal, by first name.

Which sort? You may either use one of the sorts we used in class (bubble sort or selection sort) or use the standard library sort function. If you use the system sort, you will have to define a less-than operator function for your Student type. See STL sort for arrays for how to use the Standard Template Library (STL) sort, including how to define the less-than operator.
