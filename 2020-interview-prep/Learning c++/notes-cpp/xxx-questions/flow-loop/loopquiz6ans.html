  1. Write a loop which reads integers and print their values, one per line.
    Answer: 
        int x;
    while (cin >> x) {
        cout << x << endl;
    }
  2. Write a loop which reads integers and prints the minimum value at the end of the input (EOF).
    Answer: 
        int x;
    int min;
    cin >> min;  // start minimum with first value. 
    while (cin >> x) {
        if (x < min) {
            min = x;
        }
    }
    cout << "mininmum = " << min << endl;
  3. Write a loop which prints all the odd numbers from 1 to 1,999,999
    Answer: 
        for (int i = 1; i <= 1999999; i+=2) {
        cout << i << endl;
    }
  4. Write a loop which reads two floating point numbers and prints this pair of numbers and their average (labeled), on one line. For example, for the input 7.1 1.9 the output would look like this (don't worry about the actual number of digits to the right of the decimal point): 
        7.1 1.9 Average = 4.5
    Answer: 
        float x, y;
    while (cin >> x >> y) {
        cout << x  << " " << y << "Answer = " << (x+y)/2.0 << endl;
    }

  6. Read one number from the input. Print a "square" of "*"s, the length of whose edge is the input number. This one number describes both the number of rows and columns. For example, if the input is 3, the output would look like: 
        ***
    ***
    ***
    Answer: 
        int side;
    cin >> side;
    for (int row=0; row<side; row++) {
        for (int col=0; col<side; col++) {
            cout << "*";  // start new line at end of row
        }
        cout << endl;
    }
