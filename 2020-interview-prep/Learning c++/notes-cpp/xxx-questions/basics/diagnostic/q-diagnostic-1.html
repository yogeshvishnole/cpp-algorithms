C++: Diagnostic Test

Name ______________________

  1. What is printed by this statement? 
       cout << 7 / 2;
    Output = ____________________________ Answer: 3 (integer division)
  2. What is printed by this statement? 
        cout << 7 % 2;
    Output = ____________________________ Answer: 1 (remainder after division)
  3. What is printed by this statement? 
        cout << 7.0 / 2;
    Output = ____________________________ Answer: 3.5 (floating-point division)
  4. What is printed by this statement? 
        cout << 4+3 * 4-3;
    Output = ____________________________ Answer: 13 Multiplication has higher precedence than addition and subtraction, so this is the same as 4+(3*4)-3
  5. What is printed by these statements? 
       int x=1, y=2;
   if (x != y) {
       cout << "www";
   } else {
       cout << "ppp";
   }
    Output = ____________________________ Answer: www
  6. What is printed by these statements? 
       int x=1, y=2;
   if (x < y || x > y) {
       cout << "www";
   } else {
       cout << "ppp";
   }
    Output = ____________________________ Answer: www
  7. What is printed by these statements? 
       int x=1, y=2;
   if (x < y && x > y) {
       cout << "www";
   } else {
       cout << "ppp";
   }
    Output = ____________________________ Answer: ppp (both sides of the && can't be true).
  8. What is printed by these statements? 
       int x=1, y=2;
   if (x < y) {
       if (x * 2 == x + 1) {
           cout << "www";
       } else {
           cout << "---";
       }
   } else {
       cout << "ppp";
   }
    Output = ____________________________ Answer: www
  9. What is printed by these statements? 
       int x=1, y=2;
   if (x == 44) {
       cout << "www";
   } else if (x < y) {
       cout << "ppp";
   } else {
       cout << "bbb";
   }
    Output = ____________________________ Answer: ppp
  10. What is printed by these statements? 
       int n = 0;
   while (n < 7) {
      cout << n << " ";
      n++;
   }
    Output = ____________________________ Answer: 0 1 2 3 4 5 6
