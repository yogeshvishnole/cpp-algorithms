C++: Diagnostic Test - Data Structures - Answers

Name ______________________

The purpose of this test is to get a rough idea of whether you have the basic skills to start working in the Data Structures course. You must be comfortable with the basics of one-dimensional arrays and functions. The course will review these, as well as many other features, and will cover more about these features. If you haven't used C++ in a long time, some of the questions may be difficult. If you do poorly on this test, but feel that everything will come back soon, that's ok.

This test will not count in any way toward a grade -- it's only to let you, and me, know about your current skills.

  1. What is printed by this statement? 
    cout << 9 / 5;
    Output = ____________________________ Answer: 1 (integer division)
  2. What is printed by this statement? 
    cout << 9 % 5;
    Output = ____________________________ Answer: 4 (remainder after integer division)
  3. What is printed by these statements? 
    int x=1, y=2;
if (x < y || x > y) {
    cout << "aaa";
} else {
    cout << "bbb";
}
    Output = ____________________________ Answer: aaa
  4. What is printed by these statements? 
    int x=1, y=2;
if (x < y) {
    if (x * 2 == x + 1) {
        cout << "ddd";
    } else {
        cout << "eee";
    }
} else {
   cout << "fff";
}
    Output = ____________________________ Answer: ddd
  5. What is printed by these statements? 
    int n = 0;
for (int x=1; x<4; x++) {
    n = n+x;
}
cout << "n = " << n;
    Output = ____________________________ Answer: n = 6 (ie, the sum 1+2+3)
  6. What is printed by these statements? 
    int a[] = {25, -3, 6, 0, -3, 6};
int i = 2;
cout << "a[" << i << "] = " << a[i];
    Output = ____________________________ Answer: a[2] = 6. Aray subscripts start at 0.
  7. What is printed by these statements? 
    int a[] = {25, -3, 6, 0, -3, 6};
int n = 0;
for (int i=0; i<6; i++) {
     if (a[i] < 0) {
         n++;
     }
}
cout << "n = " << n;
    Output = ____________________________ Answer: n = 2. This counts the number of negative values in the array.
  8. For the next questions assume these function definitions. 
       void f(int n){
      for (int i=0; i<n; i++) {
         cout << "f";
      }
   }
   
   void h(int m){
      cout << "h";
      f(m-1);
      cout << "h";
      return;
   }
    1. _______________ f(3); What does this print? Answer: fff
    2. _______________ h(3); What does this print? Answer: hffh
  9. Write a function with header 
       int sum(const int nums[], const int n)
    This function should add the first n values in the array nums and return that sum.
    Answer: 
    int sum(const int nums[], const int n) {
    int total = 0;
    for (int i=0; i<n; i++) {
        total = total + nums[i];
    }
    return total;
}
