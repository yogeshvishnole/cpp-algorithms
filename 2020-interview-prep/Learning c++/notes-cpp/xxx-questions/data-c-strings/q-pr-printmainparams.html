C++: Exercise - Print main Parameters

Name __________________________________

There are two legal prototypes for main. The parameterless version is the most common. The system may still pass parameters to this main, but they will be ignored.

int main()

The second form is used if a programmer wants to use the parameters on the command line.

int main(int argc, char* argv[])  // Using array notation.
int main(int argc, char** argv)   // Same as above using pointers.

For example, assume a program, myProg, is run with this command line.

myProg this is a test

When the main function in myProg is called, two parameters are passed to it.

  1. argc (argument count) is the integer count of the number of parameters including the filename that is being run. In the example above the value of argc would be 5.
  2. argv (argument values) is an array of pointers to c-strings. In the previous example, the array would be {"myProg", "this", "is", "a", "test"}.

Programming - Print main Parameters

Write a main function that prints its parameters, one per line. For this function, you must write the program without using subscripts, ie, you must use the char** form of the second parameter.
