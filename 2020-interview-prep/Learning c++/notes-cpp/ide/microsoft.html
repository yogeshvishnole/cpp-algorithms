C++ Notes: IDEs - Microsoft Visual C++ 6.0

Creating a new, single source file

  • Choose File : New menu item.
  • Click the Files tab; don't use the Projects tab.
  • In this dialog, do the following.
    • Select C++ Source File from the list.
    • Enter a file name ending with ".cpp" in the text field at the right.
    • Enter, or browse with that button, a directory.
    • Click OK.
  • Enter a minimal main program.
  • Save the program.
  • Choose Build : Compile ....cpp menu item.
  • Click Yes in response to the dialog asking if a "default project workspace" should be created.
