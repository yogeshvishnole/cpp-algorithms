Part II.B - Writing Programs (35 points)

  1. (5 points) Read in a sequence of numbers and print them out in reverse. Do not use any functions for this problem. You will have to use an array.
  2. (10 points) Write a function, countGreater, to count the number of values in an array that are larger than a specified floating=point value. The prototype is
       int countGreater(float val, float b[], int n);
    where b is an array of n floating-point numbers b[0]..b[n-1], and val is the value you will compare against each of the array values. Example call in the main program:
       float nums[] = {3.5, 1.0, 1.5};
   float wips[] = {2000., 300., -555, 0.0, 1.9999};
   int count = countGreater(1.0, nums, 3);
   cout << count;  // would print 1
   count = countGreater(-33.3, wips, 5);
   cout << count;  // would print 4
  3. (10 points) Write a function to compute the average (sum of all values divided by the number of values) of all the floating-point values in an array. The function prototype is:
       float computeAvg(float b[], int n);
    where b is an array of n floating-point values b[0]..b[n-1]. The function should compute the average value in the b array and return that value. Remember how you found the maximum value when reading values from cin with a while loop. Use the same idea but go thru the array with a for loop. If there are no values in the array, return 0.0 . Remember, this function will not do any I/O. Example call in the main program:
       float nums[] = {3.5, 1.0, 1.5};
   float wips[] = {2000., 300., -555, 0.0, 1.9999};
   float av = computeAverage(nums, 3);
   cout << av;  // would print 2.0
   av = computeAverage(wips, 5);
   cout << av;  // would print 349.4
