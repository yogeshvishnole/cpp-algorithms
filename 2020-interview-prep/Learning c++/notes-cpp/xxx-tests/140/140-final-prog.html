CMIS 140: Final Exam - Programming

Name __________________________________

There are five programming problems, and for each of them you have the choice of one of two problems. Three of the programs are identical to what you have done before. You may use arrays for all of these problems and do not have to use pointers. This is a closed book exam.
  1. (5 points) Write one of:
    1. Write a myStrlen function which is the same as the library strlen function - it returns the length of the string. Assume the prototype is 
          int myStrlen(char a[]);
    2. Write a function which shortens a string to n characters. If the string is shorter than n, the function should not change the string. Assume the prototype is 
          int truncate(char a[], int n);
  2. (10 points) Write one of:
    1. Write the myStrcat function which is the same as the library strcat function (concatenates b on the end of a). Assume the prototype is 
          void myStrcat(char a[], char b[]);
    2. Write a function to add extra blanks to the right end of a string to make it at least length width. If the string is already width characters or longer, do not change the string. Assume the prototype is 
          void padRight(char a[], int width);
  3. (10 points) Write one of:
    1. Write a function to remove all leading blanks from the left end of a string. Assume the prototype is 
          void trimLeft(char a[]);
    2. Write a function to add extra blanks to the left end of a string to make it at least length width. If the string is already width characters or longer, do not change the string. Assume the prototype is 
          void padLeft(char a[], int width);
  4. (10 points) Write one of:
    1. A main program that reads a number n, and prints a triangle made of stars (*) which has a height and width of n that looks like this for n=5: 
                  *
           **
          ***
         ****
        *****
    2. A main program that reads a number n, and prints a hollow square made of stars (*) which has a height and width of n that looks like this for n=5: 
              *****
        *   *
        *   *
        *   *
        *****
  5. (15 points) Write one of:
    1. Write a main program to read (in a loop) names into two variables: firstName and lastName. Write out the names as last, first. Before you print them, you must capitalize the first character, and make the other characters lowercase. You may use the library functions such as toupper and tolower which work on single characters.
      Example input and output:
      Input: Mickey Mouse Output: Mouse, Mickey Note: switch with ", " between.
      Input: mickey mouse Output: Mouse, Mickey Note: capitalize first character.
      Input: mIcKeY MoUse Output: Mouse, Mickey Note: make remaining characters lowercase.
    2. Write a main program that reads many words one-by-one in a loop. Print the alphabetically last word (according the strpcmp). You will find the library functions strcmp and strcpy useful. This is basically the same algorithm as used for finding the maximum of a series of integers, but there are a number of changes needed for strings. You may assume that the smallest possible value for a string is "" (the empty string with no characters in it).
