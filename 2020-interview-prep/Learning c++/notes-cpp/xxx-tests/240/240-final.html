Programming

  1. Given the following 
    struct tree_node {
    tree_node *left;   // left subtree has smaller elements
    tree_node *right;  // right subtree has larger elements
    int data;
};
    Rewrite the above struct as a class with public fields. Write a default (no parameter) constructor that does something reasonable.
  2. (15 points) Write one of the following functions. If you write more than one, you will receive the number of points of the better one.
    1. Write a program to read in words and print them in reverse order. You may use any library classes or functions. You may not used a fixed-size array.
    2. Write a program to create a four function (+, -, *, /) RPN (Reverse Polish Notation) floating-point calculator. To convert a char * string to float, use the function atof. The program should print the top of the stack after every operation. No error checking is necessary. You may use any library classes or functions.
