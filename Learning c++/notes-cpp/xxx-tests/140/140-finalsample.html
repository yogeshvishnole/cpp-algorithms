  8. Write a function to search an array for a particular value. The prototype is: 
       int find(int val, int a[], int size);
    This will search a[0]...a[size-1] for val. If it finds and element that is equal to val, the function returns the subscript of the element that has that value. If the value isn't in the array, the function returns -1. For example 
       int xyz[] = {12, 1, 8, 7};
   int zz[5] = {33, 9, 234, 25, 14};
   . . .
   cout << find(8, xyz, 4); // prints 2
   cout << find(3, xyz, 4); // prints -1 ie, not found
   cout << find(8, zz, 5);  // prints -1 ie, not found
   cout << find(14,zz, 5);  // prints 4
