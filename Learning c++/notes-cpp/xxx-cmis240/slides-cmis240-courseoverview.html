C++: Simple Types

  • Integers
    • int - This is the most commonly used integer type.
    • short
    • long
    • char - Characters are technically an integer type.
    • The signed/unsigned attributes may also be specified.
  • Floating-point
    • float - This is the standard floating-point type.
    • double
    • long double
  • bool - True or false.

C++: Structured Types

  • Character Strings - two fundamental types
    • C-strings - Zero-terminated array of chars.
    • string - type defined as a class in the STL.
  • Arrays
    • Elements accessed by subscript starting at 0.
    • All elements are the same type.
    • Fixed size
  • Structs
    • Each element has a separate name.
    • Each element may have a separate type.
    • Fixed size.
  • Classes
    • Like structs, but also allow functions.
  • Pointer and reference types
  • Other - Union (Overlapping storage) will not be covered.

C++: Expressions

  • Precedence - Each operator has a precedence
  • Numeric operators
    • + - * / % -- ++
  • Comparison operators
    • < <= == != >= >
  • Boolean operators
    • && || !
    • ?:
  • Pointer and member access operators
    • * & [] -> .
  • Bit operators
    • << | & ^ >> ~
  • Others
    • :: casts new delete sizeof etc
