C++: Programming - formatName Function

Name _______________________________

Write a funtion using C-strings which takes a first name, last name, and middle name. It combines these as last name, comma, space, first name, comma, space, middle name. The prototype is.

   void formatName(char result[], char first[], char middle[], char last[]);

If the middle name has no characters in it, the comma after the first name should be omitted. If either the first or middle name is only one character long, it should have a period added to it. For example.

You may use the standard C-string functions (strlen, strcat, strcpy, strcmp). Write only the formatName function (no main program). Do not do any I/O in the function.

Example

    char aName[100];
    char anotherName[200];
    formatName(aName, "D", "B", "Duck");
       // aName is set to "Duck, D., B." (period after single letters)
    formatName(aName, "Michael", "", "Maus");
       // aName is set to "Maus, Michael" (no comma after first name)
    formatName(anotherName, "Alpha", "Beta", "Jones");
       // anotherName is set to "Jones, Alpha, Beta"
