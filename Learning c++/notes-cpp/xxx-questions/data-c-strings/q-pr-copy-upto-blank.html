C++: Programming - copyUptoBlank Function

Name _______________________________

Write a function which takes two C-string parameters. It copies characters from the second parameter into the first parameter up to a blank or the end of the string. It also returns the length that was assigned to the the first parameter. The prototype looks like this.

   int copyUptoBlank(char dest[], char source[]);

You may not use the standard C-string functions. Write only the copyUptoBlank function (no main program). Do not do any I/O in the function.

Examples

    char someString[100];
    char anotherString[200];
    int n;
    n = copyUptoBlank(someString, "This is a test");
         // someString will contain "This" and n will be 4.
    n = copyUptoBlank(anotherString, "Hello");
         // anotherString will contain "Hello" and n will be 5.
    strcpy(someString, "x");
    cout << copyUptoBlank(anotherString, someString);
         // will print 1 and anotherString will contain "x".
