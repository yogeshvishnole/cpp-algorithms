C++: Programming: cstringArrayToVector() function

Name __________________________________

There are two equivalent forms that parameters are passed to main from the command line.

int main(int argc, char* argv[])  // Using array notation.
int main(int argc, char** argv)   // Same as above using pointers.

Write a function that takes a parameter list similar to that passed to main and returns a pointer to a vector of strings. argc is the number of elements in the argv array. argv is an array of pointers to C-strings. 

  vector<string>* cstringArrayToVector(int argc, char** argv);

Hint

You will have to dynamically allocate the vector inside of this function. If you declared the vector to be a local variable in the function, its destructor would be called and it would be popped off the stack when the function returned.
