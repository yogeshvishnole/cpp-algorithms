(5 points) Write a program to read in a number and print out one of three messages, depending on the value of the number.

  • If the number is less than 72, print "Too cold".
  • If the number is between 72 and 78 (inclusive), print "Just right".
  • And if the number is above 78, print "Too hot".

Write the entire program (includes, main, etc). There are several ways to write this program.
