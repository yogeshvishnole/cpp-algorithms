Given this declaration.

struct Node {
    Node* next;
    Node* prev;
    float data;
};

Write a function that uses the above definition of Node and which appends one list to the end of another. The prototype is. 

Node* listcat(Node* list1, Node* list2);

the function listcat appends list2 to the end of list1.

Return value. This returns the address of the first element of the resulting list (ie, list1). If list1 is NULL, return list2

Hint: All this does is replace the NULL link at the end of list1 with the address of the first element of list2, and the NULL link at the beginning of list2 with the address of the last element of list1. You will need a loop to find the end of list1. Do not make copies of any nodes.

//============================================================= listcat
Node* listcat(Node* list1, Node* list2) {
    if (list1 == NULL) return list2;
    if (list2 == NULL) return list1;
    
    Node* p = list1;  // p will point to last element of list1.
    while (p->next != NULL) {
        p = p->next;
    }
    p->next = list2;  // Point last elem of list1 to list2.
    list2->prev = p;  // Point first elem of list2 to last elem of list1.
}
