C++: Exercise - List average function

Name __________________________________

Given this declaration.

struct Node {
    Node* next;
    Node* prev;
    float data;
};

Write a function that uses the above definition of Node and which computes the average value in the list. The prototype is.

    float avg(Node* flist);

This function computes the average of the values in the list flist and returns that value. If the list is empty, it returns zero.

Write only the function, not a main program, in the space below.
