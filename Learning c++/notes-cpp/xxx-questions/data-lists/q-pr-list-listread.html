C++: Exercise - listread

Name __________________________________

Write a function, listread, which reads floats from cin and puts them in a linked list. Return the address of the first element of the list. If there is no input, the function should return NULL. Use these definitions:

struct Node {
    Node* next;
    float data;
};

The function prototype is.

Node* listread();  // Returns pointer to first element of list

Hints. Here are some things you should be sure your code does.

  • Save the pointer to the first node that is allocated so it can be returned at the end. You'll probably need to declare at least two pointers, one to hold the address of the first node, and one to point to the last node in the list.
  • Be sure to set the next pointer to NULL for the last entry (or every entry then overwrite it).
