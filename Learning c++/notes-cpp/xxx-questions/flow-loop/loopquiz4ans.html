  5. What is the output given input: 5 3 -6 8 -2 1 
       // Loop written in "Sentinal" style.
   int number;
   int sum = 0;
   bool nonNegative = true;
   bool notEOF = true;
   while (nonNegative && notEOF) {
       if (cin >> number) {
           if (number < 0) {
               nonNegative = false;
           }else{
               sum = sum + number;
           }
      }else{
           notEOF = false;
      }
   }
   cout << sum << endl;

    Answer: 8
    This reads and adds numbers until it gets to the end of file (EOF) or a negative number. So here it only reads and adds the first two numbers.
  6. What is the output given input: 5 3 -6 8 -2 1 
       int number;
   int sum = 0;
   while ((cin >> number) && (number >=0)) {
       sum += number;
   }
   cout << sum << endl;
    Note: the above loop is like the previous problem, but makes a direct test instead of setting sentinals to test later. Which is more readable?
    Answer: 8
    This (same as preceding question) reads and adds numbers until it gets to the end of file (EOF) or a negative number. So here it only reads and adds the first two numbers.

  7. Write a loop to print all numbers from 1 to 100, one per line, with a blank line after every five lines.
    Answer: 
        int lineCount = 0;   // counter for output lines.
    for (int i=1; i<=100; i++) {
        cout << i << ' ';
        lineCount++;
        if (lineCount % 5) == 0) {
            cout << endl;
        }
    }
    Counting the output lines with a separate counter is a typical idiom, but because lineCount and i are both being incremented together, we don't need lineCount in this particular program.
