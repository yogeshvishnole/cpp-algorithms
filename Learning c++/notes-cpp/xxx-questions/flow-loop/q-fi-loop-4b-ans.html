  1. What is the output given these six input values: 5 3 -6 8 -2 1 ? 
    int n;
while (cin >> n) {
    cout << n << " ";
}
    Output = ______________________________________________
    Answer = 5 3 -6 8 -2 1
    This just echos the input numbers.
  2. What is the output given these six input values: 5 3 -6 8 -2 1 ? 
    int n = 7;
int x = 0;
while (cin >> n) {
    x++;
}
cout << x;
    Output = ______________________________________________
    Answer = 6
    This simply counts the input numbers. x isn't a good variable name, but I didn't want to give away the purpose.
  3. What is the output? 
    int n = 4;
for (int i=n; i>0; i--) {
    cout << i << " ";
}
    Output = ______________________________________________
    Answer = 4 3 2 1
    Counts down.
  4. What is the output given these five input values: 5 3 8 -2 10 ? 
    int number;
int sum = 0;
while ((cin >> number) && (number > 0)) {
    sum = sum + number;
}
cout << sum << endl;
    Output = ______________________________________________
    Answer = 16
    The sum of 5+3+8. The while is terminated when the while clause becomes false after reading the -2.
  5. Rewrite this code using a for loop. 
    int n = 0;
while (n < 9) {
    if (n%2 == 0) {
        cout << n  << " is even" << endl;
    } else {
        cout << n  << " is odd" << endl;
    }
    n++;
}











    Answer 
    for (int n=0; n<9; n++) {
    if (n%2 == 0) {
        cout << n  << " is even" << endl;
    } else {
        cout << n  << " is odd" << endl;
    }
}
    This isn't exactly the same because the declaration of n was moved inside the scope of the loop, so it won't be accessible later. However, this is the typical equivalent.
