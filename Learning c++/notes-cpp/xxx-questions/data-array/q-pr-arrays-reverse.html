C++: Arrays: Programming Problem: Reverse Input

Name __________________________________

(5 points) Read in a sequence of floating-point numbers terminated by an end-of-file and print them out in reverse. Assume there will not be more than 100,000 numbers. Do not use any functions for this problem. Hint: Read the numbers into an array and them print them.

Answer: 
// arrays/reverse-input.cpp - Reverses order of numbers in input.
#include <iostream>
using namespace std;

int main() {
    //--- declare array and number of elements in it.
    float a[100000];
    int n;
   
    //--- read numbers into an array
    n = 0;
    while (cin >> a[n]) {
        n++;
    }
   
    //--- print array in reverse order
    for (int i=n-1; i>=0; i--) {
        cout << a[i] << endl;
    }
   
    return 0;
}//end main
