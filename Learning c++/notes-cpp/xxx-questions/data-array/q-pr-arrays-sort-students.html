C++: Arrays: Programming Problem: Sort Students

Name __________________________________

(5 points) Read lines of information about students and their scores on on a test. Sort this information by score, and print a new list of scores and students with this information.

Specifications and Implementation.
  • Input data. A few sample lines of the input data might look like the following, with a test score followed by a first name followed by a last name. 
       74 JOHN SMITH
   95 JILL SMITH
   87 BILLY JONES
  • The maximum number of students that will be read, until an EOF, is 1000 (ie, the maximum array size).
  • Types. The score is an int, and each of the names should be a string. Do not use C-strings (char arrays) for the names because they are difficult to use in arrays.
  • Parallel arrays or structs. This data will have to be read into arrays and sorted. There are two general approaches.
    • Three arrays can be used, one for scores, one for first names, and one for last names. If this approach is used, any exchanges in the sorting process must exchange the corresponding elements the all three arrays.
    • A struct can be defined that holds all three values. For example, 
      struct Student {
          string first_name;
          string last_name;
          int    score;
       };
      Then a single array of Student structs can be used. This is a better approach, but use parallel arrays if you don't feel comfortable with structs yet.
  • Sort algorithm. Use either a bubble sort of a selection sort. The selection sort is a little more efficient because the number of exchanges is smaller.
  • Implementation strategy. Try using good iterative development techniques. Implement the program in stages. Compile and test each stage before continuing. For example,
    1. Write a main that does nothing except return.
    2. Add some arrays and read into them.
    3. Write out the arrays.
    4. Once you can read and write the arrays correctly, write a sort function.
  • Sample output for the sample input. 
       74 JOHN SMITH
   87 BILLY JONES
   95 JILL SMITH
    Of course, you should test it with much more data.
