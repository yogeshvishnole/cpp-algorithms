C++: swap struct Programming Problem

Name __________________________________

(10 points) Write a function, swap, that exchanges the values of the two reference parameters. Assume the following type has been defined. 
struct Student {
    int    student_id;
    string student_name;
};
Use the following prototype. 

   void swap(Student& a, Student& b);
Here is an example call. 
Student bob;
Student betty;

bob.student_id = 12345;
bob.student_name = "Bob Smith";

betty.student_id = 98765;
betty.student_name = "Betty Boop";

cout << bob.student_name;  // prints Bob Smith

swap(bob, betty);

cout << bob.student_name;  // prints Betty Boop
