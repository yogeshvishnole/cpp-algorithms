  1. (25 points) Write the corresponding .cpp file that implements this class declaration. 
    // Person.h
#ifndef PERSON_H
#define PERSON_H

#include <string>
using namespace std;

class Person {
private:
    string _name;  // pointer to name
    int    _id;    // ID
public:
    Person(char* n, int i);  // constructor
    Person(string n, int i);  // constructor
    Person(const Person& p); // copy constructor
    ~Person();               // destructor
    char* getName() const;
    int   getID()   const;
    bool operator==(const Person p) const;
};
#endif // PERSON_H
    • Constructor: The constructor is passed a pointer to a character string. You must allocate new memory and copy the name to that new copy. Assume that it is always passed a pointer to a valid name. This will involve calling strcpy.
    • A copy contructor is needed to make a deep copy, ie, a copy of the name field has to be made.
    • A destructor is needed to deallocate the memory that was allocated for the name.
    • The == operator definition will have to call strcmp.
