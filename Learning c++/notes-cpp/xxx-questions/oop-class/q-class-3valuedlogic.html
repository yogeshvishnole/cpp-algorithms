C++: Class for 3-valued logic

Some languages, most notably SQL, use three-valued logic: true, false, and unknown.

Write a class, L3, for 3-valued logic

It should have some of the following characteristics.
  • Override assignment so that bools can be assigned to an L3 object.
  • Override && to work with L3s and bools. The operators || and ! should also be overridden, but writing && is sufficient.
  • Define three new constants: L3::TRUE3, L3::FALSE3, and L3::UNKNOWN3.
  • Define three member functions: 
       bool isTrue();
   bool isFalse();
   bool isUnknown();
